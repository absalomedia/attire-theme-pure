# Attire-Pure

A Yahoo Pure ([http://purecss.io/](http://purecss.io/)) theme for Codeigniter v3.0 using Attire, with Font Awesome included.

## Installation

With composer:

    composer require abm/attire-theme-pure dev-master

## Page Templates

This theme comes with different page layouts.

* Blog 
* Landing Page
* Pricing Table
* Sidebar
