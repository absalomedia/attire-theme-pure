# Attire-Pure

A Yahoo Pure ([http://purecss.io/](http://purecss.io/)) theme for Codeigniter v3.0 using Attire, with Font Awesome included.

[![Codacy Badge](https://api.codacy.com/project/badge/Grade/a4a180f7bb83400aa81c9a4546c5233b)](https://www.codacy.com/app/media/attire-theme-pure?utm_source=github.com&amp;utm_medium=referral&amp;utm_content=absalomedia/attire-theme-pure&amp;utm_campaign=Badge_Grade) [![StyleCI](https://styleci.io/repos/53823390/shield)](https://styleci.io/repos/53823390)

## Installation

With composer:

    composer require abm/attire-theme-pure dev-master

## Page Templates

This theme comes with different page layouts.

* Blog 
* Landing Page
* Pricing Table
* Sidebar
